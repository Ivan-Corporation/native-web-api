# Web API Playground


Playground for native browser Web APIs. Meme-y, developer-friendly, and made for short episodes.


## Setup


1. `npm install`
2. `npm run dev`


## Add a new API demo


1. create `src/hooks/useSomething.ts`
2. create `src/components/SomethingDemo.tsx`
3. export from `src/data/apis.ts` with emoji + description
